About Me

Hi, I'm Himanshu Sharma, currently pursuing a Bachelor's in Internet of Things (IoT) at Dayalbagh Educational Institute, Agra. I'm passionate about cloud computing, web development, and the intersection of hardware and software systems.

I specialize in building full stack IoT systems that connect embedded hardware to cloud platforms with realtime web interfaces. i enjoy working at the intersection of embedded tech , web development and cloud computing .

Some notable projects I've worked on include:

  • A device management web app for LoRa-based IoT devices
  • A local chatbot using Django and Ollama
  • A mood-based music streaming platform using natural language processing

I've also built interactive systems using Arduino, ESP boards, MQTT, and Tailwind CSS to deliver responsive, engaging user experiences. Currently, I'm seeking an internship opportunity where I can apply my skills, collaborate on innovative projects, and continue to grow as a developer in the cloud and web tech space.
